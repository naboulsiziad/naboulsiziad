

# Ziad Naboulsi — Videographer / Editor Portfolio

A minimal, premium portfolio website with a dark cinematic aesthetic, bold typography, and subtle animations. Designed for maximum impact with generous whitespace and a confident project grid.

---

## Design System

**Visual Direction:**
- Deep charcoal/near-black background with light text
- Bold geometric sans-serif typography (strong headlines)
- Generous spacing and breathing room
- Subtle fade-in animations on scroll
- Smooth hover states on interactive elements

**SEO Defaults:**
- Unique page titles and meta descriptions per page
- Open Graph image and metadata for social sharing
- Semantic HTML structure

---

## Pages & Features

### 1. Home Page
- **Header navigation:** Work · About · Contact + "Request a Quote" CTA button
- **Hero section:** 
  - Name, tagline, two action buttons (Watch Reel / View Work)
  - Vimeo reel with **click-to-play poster thumbnail** (no autoplay for performance)
- **Selected Work mini-strip:** 3 featured projects immediately below hero (work visible before scrolling to reel)
- **Selected Clients / Collaborations:** Logo strip or styled name list near the top
- **Full Selected Work grid:** 6–9 project cards linking to individual project pages
- **Services section:** Two-column layout (Post-Production / Production capabilities)
- **About teaser:** Short bio paragraph with "More about me" link
- **CTA band:** "Let's build your next video" with contact buttons

### 2. Work Page
- **Page header:** Title + descriptive line
- **Filter bar:** All · Commercial · Social · Broadcast · Film
- **Filterable project grid:** Thumbnail cards (no autoplay), each linking to its own project page
- Clean hover effects on cards

### 3. Individual Project Pages (Dynamic Template)
- Project title
- Meta line (Role · Year · Deliverables)
- Video embed area (links to Instagram/YouTube for now, Vimeo later)
- Short project description
- Optional credits section
- **Previous / Next project navigation**
- **Clear "Back to Work" link**

### 4. About Page
- Bio paragraphs (visual storytelling focus)
- Experience timeline (Advert Communications, Solo Films, Freelance)
- Tools section (Adobe Premiere, Audition, Logic Pro, etc.)
- Gear familiarity (Sony cameras, Ronin, Aputure lighting)
- Education (BA in Communication, LIU 2022)
- Languages: Arabic (Native), **English (Professional)**

### 5. Contact Page
- Header + descriptive line
- **Response time line** (e.g., "I typically respond within 24 hours")
- Contact details: Email, WhatsApp, Location (Lebanon)
- **WhatsApp quick button** (opens WhatsApp directly)
- Visual contact form: Name, Email/WhatsApp, Project type dropdown, Deadline, Message
- Form is visual-only for now (no backend)

Placeholder projects ready for your real content:

## Technical Notes

- Responsive design (mobile-friendly navigation)
- React Router for page navigation
- Project data stored in a local file (easy to update with your real content)
- Animations using CSS transitions and scroll-triggered effects
- SEO: React Helmet or meta tags for page-specific titles/descriptions
